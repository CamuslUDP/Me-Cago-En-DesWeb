Información sobre la aplicación y su equipo

Información sobre la aplicación

RappiBet es una aplicación web ficticia que simula la experiencia de un casino en línea con ruleta europea. El propósito de este proyecto es aplicar conocimientos del curso Desarrollo Web y Móvil (CIT2008), poniendo en práctica HTML semántico, CSS responsivo y criterios de diseño visual.

El equipo de desarrollo

  • Álvaro Acevedo — Escuela de Informática y Telecomunicaciones, UDP
  • Matías Rojas — Escuela de Informática y Telecomunicaciones, UDP

Ambos integrantes colaboraron en el diseño, la estructura y el desarrollo del sitio.

Tecnologías utilizadas

Este proyecto fue desarrollado exclusivamente con:

  • HTML5 semántico
  • CSS3 con diseño responsivo
  • Imágenes optimizadas para la web

No se utilizaron frameworks ni JavaScript, respetando las restricciones técnicas del trabajo.

Aclaración académica

Este sitio web es un proyecto académico y no representa una plataforma real de apuestas. Todo el dinero, estadísticas y resultados mostrados son ficticios y se usan únicamente con fines educativos.
